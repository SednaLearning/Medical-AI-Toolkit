# Medical-AI-Toolkit
Dieses Repository enthält meine Werkzeuge für die KI-gestützte Prüfung regulatorischer und klinischer Dokumente.
## Ziele

- Wiederverwendbare Prompts entwickeln
- Eigene Checklisten zu Regelwerken pflegen
- Vorlagen für Reviews und Prüfberichte verwalten
- Beispiele dokumentieren
- Verbesserungen versionieren

## Struktur

- Regelwerke
- Prompts
- Vorlagen
- Beispiele
- Notizen
